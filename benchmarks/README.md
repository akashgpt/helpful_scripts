# benchmarks

## Overview

This folder stores compact benchmark references: known-good scripts, concise result summaries, input templates, and conclusions from cluster tests. It is meant for reusable knowledge, not raw log dumps.

Use this folder before creating a new benchmark script or choosing production settings. For ALCHEMY and QMD work, these files preserve what worked, what failed, and why.
